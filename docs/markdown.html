Markdown教程

标题

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

以上标题分别对应着HTML中的<h1>、<h2>、<h3>、<h4>、<h5>、<h6>标签。

强调

加粗

**这句话会被加粗**
__这句话会被加粗__

加粗对应着HTML中的<strong>标签。

###斜体

*这句话会被斜体显示*
_这句话会被斜体显示_

斜体对应着HTML中的<em>标签。

无序列表

* Item 1
* Item 2
  * Item 2a
  * Item 2b

或者

- Item 1
- Item 2
  - Item 2a
  - Item 2b

或者

* Item 1
* Item 2
  - Item 2a
  - Item 2b

或者

- Item 1
- Item 2
  * Item 2a
  * Item 2b

###有序列表

1. Item 1
1. Item 2
1. Item 3
   1. Item 3a
   1. Item 3b

图片

![Logo](/images/logo.png)

链接

[四海汉得](http://eco.hand-china.com)

引用

> 这是一段引用

或者

<blockquote>
    这是一段引用
</blockquote>
注意：Markdown可兼容HTML语法，所以可以在Markdown中通过相应的HTML标签来实现类似或者更复杂功能。

行内代码

`inline code`

代码块

​```
public class Hello {
    public static void main() {
        System.out.println("Hello World!");
    }
}
​```

或者

​```java
public class Hello {
    public static void main() {
        System.out.println("Hello World!");
    }
}
​```

任务列表

- [x] 已完成任务1
- [x] 已完成任务2
- [x] 已完成任务3
- [ ] 未完成任务

表格

属性1 | 属性2
---|---
值1 | 值2
值3 | 值4

Markdown内置的表格标签只能支持简单表格，对于复杂表格可以使用HTML标签实现：

<table>
    <thead>
        <tr>
            <th>属性1</th>
            <th>属性2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>值1</td>
            <td>值2</td>
        </tr>
        <tr>
            <td>值3</td>
            <td>值4</td>
        </tr>
    </tbody>
</table>
